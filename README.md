# 关于 SSR 

1. 使用 [inoreader](https://www.inoreader.com/)订阅,我的订阅为`Inoreader Feeds 20220121.xml` 文件
2. 使用 [RSS Feed Reader](https://chrome.google.com/webstore/category/extensions?hl=zh-CN) 谷歌插件 订阅
